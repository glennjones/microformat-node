Just a name (h-card parsing test)

Frances Berriman

The expected JSON output:


[
    {
        "type": ["h-card"],
        "properties": {
            "name": ["Frances Berriman"]
        }
    }
]

Assertions for integration tests:

Test Result Comment
name[0] IsEqualTo('Frances Berriman') The name of the h-card

Just a hyperlink (h-card parsing test)

Ben Ward

The expected JSON output:


[
    {
        "type": ["h-card"],
        "properties": {
            "name": ["Ben Ward"],
            "url": ["http://benward.me/"]
        }
    }
]

Assertions for integration tests:

Test Result Comment
name[0] IsEqualTo('Ben Ward') The name of the h-card
url[0] IsEqualTo('http://benward.me/') The url of the h-card

An extended description (h-card parsing test)

Mitchell Baker (@MitchellBaker) Mozilla Foundation

Mitchell is responsible for setting the direction and scope of the Mozilla Foundation and its activities.

Strategy and Leadership

The expected JSON output:


[
    {
        "type": ["h-card"],
        "properties": {
            "category": ["Strategy", "Leadership"],
            "name": ["Mitchell Baker"],
            "note": ["Mitchell is responsible for setting the direction and scope of the Mozilla Foundation and its activities."],
            "org": ["Mozilla Foundation"],
            "photo": ["http://blog.mozilla.org/press/files/2012/04/mitchell-baker.jpg"],
            "url": ["http://blog.lizardwrangler.com/", "https://twitter.com/MitchellBaker"]
    }
]

Assertions for integration tests:

Test Result Comment
category[1] IsEqualTo('Leadership') The secong category of the h-card
name[0] IsEqualTo('Mitchell Baker') The name of the h-card
note[0] IsEqualTo('Mitchell is responsible for setting the direction and scope of the Mozilla Foundation and its activities.') The note of the h-card
org[0] IsEqualTo('Mozilla Foundation') The org of the h-card
photo[0] IsEqualTo('http://blog.mozilla.org/press/files/2012/04/mitchell-baker.jpg') The photo of the h-card
url[1] IsEqualTo('https://twitter.com/MitchellBaker') The second url of the h-card

hreview-aggregate parsing tests

This page was design to test the parsing of hreview-aggregate. These tests are part of the micorformats 2 test suite.

  • Minimum properties
  • With item as a vcard
  • With item as a vevent

Minimum properties (hreview-aggregate parsing test)

Mediterranean Wraps - Rated: 4.5 out of 5 (6 reviews)

The expected JSON output:


{
    "items": [{
        "type": ["h-review-aggregate"],
        "properties": {
            "item": [{
                "value": "",
                "type": ["h-item"],
                "properties": {
                    "name": ["Mediterranean Wraps"],
                    "url": ["http://example.com/mediterraneanwraps"]
                }
            }],
            "rating": ["4.5"],
            "count": ["6"],
            "name": ["Mediterranean Wraps - Rated: 4.5 out of 5 (6 reviews)"]
        }
    }]
}
